Insight UBC

Project Summary

Insight UBC is a full-stack web development project.
- server built using Node.js in the Typescript language (Superset of JavaScript developed by Microsoft that has optional typing. It compiles to plain JavaScript). implementing a REST API server
- Used the express server library. The
- client-side implementation used HTML, CSS, and Javascript.
- Testing used the Mocha framework with the chai library.
